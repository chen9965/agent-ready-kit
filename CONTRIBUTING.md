# Contributing / 贡献指南

Thanks for helping make repositories easier for AI agents to work in.

感谢你帮助更多仓库适配 AI 编码代理。

## Development / 开发

```bash
npm install
npm run build
npm test
```

## Pull Requests / PR

- Keep changes focused and include tests for scanner or generator behavior.
- 保持修改聚焦，并为扫描器或生成器行为添加测试。
- Prefer bilingual user-facing text.
- 面向用户的文本优先保持中英文对照。
- Update README examples when CLI behavior changes.
- CLI 行为变化时同步更新 README 示例。
